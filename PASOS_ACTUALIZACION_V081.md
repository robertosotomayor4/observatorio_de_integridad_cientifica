# Actualización rápida a la interfaz v0.8.1

## Cuando todavía no se publicó la v0.8

1. Descomprimir `observatorio_integridad_cientifica_v0_8_1_github_ready.zip`.
2. Copiar su contenido en la carpeta local del repositorio.
3. Confirmar en GitHub Desktop los archivos nuevos y modificados.
4. Realizar commit, por ejemplo: `Actualizar interfaz a v0.8.1`.
5. Hacer **Push origin**.
6. Verificar GitHub Pages.

## Cuando la v0.8 ya está en el repositorio

Puede utilizarse `parche_visual_observatorio_v0_8_1.zip`. Copiar su contenido sobre el repositorio respetando las rutas y confirmar el reemplazo.

## Verificación mínima después de publicar

1. Buscar una revista con datos de SCImago e InCites.
2. Confirmar que la ficha muestre una sola sección cuantitativa.
3. Confirmar que dentro de ella aparezcan:
   - Scopus / SCImago en naranja.
   - Web of Science / InCites en azul.
4. Confirmar que OpenAlex figure por separado.
5. Probar la vista móvil o reducir el ancho del navegador: los dos bloques deben apilarse verticalmente.

## Base de datos

No es necesario sustituir las bases SQLite v0.8. La v0.8.1 no modifica datos ni metodología de cálculo.
